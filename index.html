MorseBot

Technologies Involed:

HTML/CSS/Javascript, JQuery, Python, MongoDB, Node.js


A group project for the IEEE Quarterly Projects at UCSD developed in Fall 2016. The project was a web application combined with a Morse straight key attached to a Raspberry Pi allowing two users to communicate with each other via Morse Code. The messages they send are securely sent to a MongoDB collection using Node.js running on the Pi. The web app also displays the latest messages sent and a chart with the frequency of dots and dashes translated and English letters translated. The project won Second place in the IEEE Fall Project Competition at UCSD.
